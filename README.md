# EventHub

EventHub adalah aplikasi sederhana untuk mengelola data event menggunakan Node.js, Express, MySQL, dan Svelte.

## Fitur

- Login Admin
- Menambah Event
- Mengedit Event
- Menghapus Event
- Menampilkan Daftar Event
- Dashboard Statistik

## Teknologi

- Svelte
- Bootstrap 5
- Node.js
- Express.js
- MySQL

## Cara Menjalankan

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
